# PlantUML Export Test Repo

This repo tests a GitHub Actions workflow that:
1) renders PlantUML (*.puml) into images, and
2) commits the generated files back into the repo.

- Sources: `diagrams/src/*.puml`
- Config: `diagrams/config/config.puml`
- Outputs: `diagrams/out/png` (and optionally `diagrams/out/jpg`)
